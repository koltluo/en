6.1 Factory Reset

Setting P0-29 to 10000 restores all parameters in the motor parameter group to their factory default values. Setting P0-29 to 10000 has the same effect as setting PP-01 to 1 for factory reset. Before executing macro operations for industrial applications, please execute: P0-29=10000.
